# Linktree PRT

Bem-vindo ao Linktree PRT, um projeto desenvolvido por Gabriel Lima para centralizar e destacar links importantes de portfólio, redes sociais e contato profissional.

## Sobre o Projeto

Este projeto foi criado com foco em design moderno, responsivo e acessível, ideal para uso como página de apresentação pessoal ou profissional. Inspirado no conceito de Linktree, permite reunir todos os seus links em um só lugar, facilitando o acesso e a divulgação.

## Funcionalidades

- **Interface Responsiva:** Adaptável a dispositivos móveis, tablets e desktops.
- **Design Moderno:** Utilização de animações, ícones e paleta de cores elegante.
- **Links Personalizados:** Botões para WhatsApp, Github, LinkedIn, Portfólio e Instagram.
- **Foto de Perfil:** Destaque visual para identidade pessoal.
- **Tema Escuro/Claro:** Alternância de tema para melhor experiência do usuário.

## Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript
- Font Awesome (ícones)

## Estrutura do Projeto

```
linktree-prt/
├── index.html
├── style.css
├── script.js
├── Foto/
│   └── FotoPerfil.jpg
```

## Como Usar

1. Clone o repositório:
   ```bash
   git clone https://github.com/GabrielLima-BD/linktree-prt.git
   ```
2. Abra o arquivo `index.html` em seu navegador.
3. Personalize os links e a foto de perfil conforme sua necessidade.
